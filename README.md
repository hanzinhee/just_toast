# Just Toast

This is just toast!

# Usage

```
    showToast(context: context, text: 'toast message');

```
- custom

```
    showToast(
      context: context,
      text: 'Custom Toast message',
      duration: const Duration(milliseconds: 1000),
      alignment: Alignment.center,
      textStyle: const TextStyle(fontSize: 20, color: Colors.amber),
      color: Colors.purple,
    );
```
